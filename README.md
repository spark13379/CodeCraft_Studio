## Estrutura do Projeto
- Frontend: React.js + Vite
- Backend: Node.js + Express
- Database: JSON local (`projects.json`)

## Como Rodar

1. **Backend**
   ```bash
   cd backend
   npm install
   node server.js
   ```

2. **Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Acesse [http://localhost:5173](http://localhost:5173) no navegador.
